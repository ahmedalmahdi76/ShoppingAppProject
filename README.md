# 🛒 ShoppingAppProject  

**ShoppingAppProject** is a final project for the **Mobile Development Flutter Course** provided by **Sprintis**.  
This mobile application is designed as a digital market platform where users can browse item categories, view offers, and manage their shopping experience. It also includes a complete **authentication system** with validation rules and multi-language support.  

---

## ✨ Features  

### 🚀 Onboarding  
- **PageView walkthrough** with three pages.  
- Each page contains a **Skip** button navigating to authentication.  
- The last page includes a **Get Started** button that also leads to authentication.

### 🔐 Authentication  
- **Sign In / Sign Up system** with full validation.  
- **Sign Up validation rules**:  
  - Full name must start with an uppercase letter.  
  - Email must follow a valid format (`example@gmail.com`).  
  - Password must be at least **6 characters**.  
  - Confirm password must match the original password.  
- **Navigation option** between **Sign In** and **Sign Up** screens using clickable text.
